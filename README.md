# 🚀 AI Resume Analyzer

An AI-powered ATS Resume Analyzer built using Python and Flask.

## Features
- Resume PDF Upload
- ATS Match Score
- Skill Extraction
- Missing Skills Detection
- Resume Feedback
- Modern Responsive UI

## Technologies Used
- Python
- Flask
- HTML
- CSS
- PDFPlumber

## Run Project

```bash
pip install -r requirements.txt
python app.py
